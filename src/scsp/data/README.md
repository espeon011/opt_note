# SCSP データセット

ここでは SCSP のデータファイルは改行区切りの文字列からなる. 

## 一様ランダム生成

各データファイルは `uniform_q{}n{}k{}-{}.txt` の形式になっており,
それぞれの中括弧には以下が入る.

1. 文字の種類の数
1. 文字列の数
1. 文字列の長さの最小値
1. 文字列の長さの最大値

ただし上記の数値はランダム生成するときにそう設定しただけで実際にデータセットがぴったりそうなっているとは限らない. 

## ヌクレオチド配列

各データファイルは `nucleotide_n{}k{}.txt` の形式になっており, 
それぞれの中括弧には以下が入る. 

1. 文字列の数
1. 文字列の長さ

## プロテイン配列


各データファイルは `protein_n{}k{}.txt` の形式になっており, 
それぞれの中括弧には以下が入る. 

1. 文字列の数
1. 文字列の長さ
